# launchd

The launchd plist is now generated by `scripts/install_agent.sh` from
`MANIFEST.txt` + `$TRUESIGHT_WORKSPACE`, rather than being a static
template. This means:

- New repos added to `MANIFEST.txt` automatically pick up WatchPaths on
  the next `install_agent.sh` run — no manual plist edit needed.
- Governors with non-default workspace roots get correct WatchPaths for
  free.

The rendered plist lands at
`~/Library/LaunchAgents/me.truesight.credential-backup.plist`. Don't
edit it by hand — it'll be overwritten the next time `install_agent.sh`
runs.

To preview what install_agent.sh would render without installing:

```bash
~/Applications/credential_vault/scripts/install_agent.sh --print-only | less
```

To uninstall:

```bash
~/Applications/credential_vault/scripts/install_agent.sh --uninstall
```
